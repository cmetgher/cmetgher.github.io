Creating a Github Blog Using Octopress

Oct 10th, 2013

With Octopress, I write my blog posts in Markdown. Octopress uses uses Jekyll that runs with Github pages.

Markdown offers a clean and simple syntax for text formating, translates fast and easily to HTML. It’s flexibility allows for outputing files into a wide array of formats, that are implicitly cross-platform.

Several third party Octopress themes are available here. There are several resources to help you get started with Octopress, such as the following:

  • Blog With Octopress and Github Pages)
  • Creating a Github Pages Blog With Octopress
  • Blogging With Octopress here and here

The documentation on the Octopress is pretty good too. Enjoy!
